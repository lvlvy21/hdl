# DailyTasksApp (SwiftUI)

已按需求创建 `apps/DailyTasksApp/DailyTasksApp.swift`，并使用你提供的代码内容。

## 在 Xcode 中新建项目并使用
1. 打开 Xcode，选择 **File > New > Project > iOS App**。
2. Product Name 填 `DailyTasksApp`，Interface 选 **SwiftUI**，Language 选 **Swift**。
3. 创建项目后，用本目录下的 `DailyTasksApp.swift` 替换默认同名文件内容。
4. 运行到 iPhone 模拟器即可。
